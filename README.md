# Telegram Markdown → DOCX Bot (ГОСТ 7.32-2017)

Telegram-бот, который принимает Markdown (текст или файл) и генерирует DOCX-документ, оформленный по ГОСТ 7.32-2017.

## Возможности

- Титульная страница с данными из YAML-заголовка
- Автоматическое оглавление (обновляется при открытии в Word)
- Форматирование: Times New Roman 14pt, полуторный интервал, отступ 1.25 см
- Заголовки, списки, таблицы, блоки кода
- Нумерация страниц (со 2-й страницы)
- Выбор типа работы: лабораторная, курсовая, отчёт по практике

## Установка

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## Настройка

Создайте файл `.env` в корне проекта:

```
BOT_TOKEN=ваш-токен-от-BotFather
```

## Запуск

```bash
python -m bot.main
```

## Использование

1. `/start` — приветствие и инструкция
2. `/help` — поддерживаемый формат Markdown
3. `/template` — скачать пример шаблона
4. Отправьте Markdown-текст или файл (.md/.txt) → выберите тип работы → получите DOCX
